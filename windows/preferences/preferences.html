App Preferences

Notifications

Track Changes

Show a notification when the track changes to another

On Playback

Show a notification when a track starts playback

On Pause

Show a notification when the track is paused

On Stop

Show a notification when the queue is finished

Only When Focused

Show any enabled notification but only when you're not using the player window

Theme

Change the look of the interface

Tray

Enabled

Toggles whether Spotify Web Player for Linux should show a tray icon?

Icon

Mix-&-Match the tray icon to your DE/Icon style

Close To Tray

Allows you to keep the window hidden whilst playing music and use the tray menu or application menu to quit.
The window will minimise if the tray icon is disabled

Close To Controller

Allows you to keep the window hidden without an indicator or application icon taking up space
Warning: MPRIS controllers have to raise Spotify Web Player for it to become unhidden

Show Application Menu

Toggles whether Spotify Web Player for Linux should hide the application menu
 Warning! Without a tray icon, you may not be able to get into the preferences window

Start on Login

Toggles whether Spotify Web Player for Linux will start on user login

Start Hidden

Toggles whether Spotify Web Player for Linux will start minimized

Store Album Art?

Toggles whether album art will be stored for caching
Note: Album Art will still be downloaded temporarily to /tmp for notifications (but deleted on reboot)

Album Art Cache Size

Choose the size of the Album Art that will be cached
Note: Larger images will take up more disk space over time

Clean Album Cache

This will remove all album art covers 		Clean Me
